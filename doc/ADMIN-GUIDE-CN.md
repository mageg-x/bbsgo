## 前言

欢迎使用彩虹BBS管理操作指南！本文档将详细介绍彩虹BBS社区论坛系统的管理后台所有功能，以及主站的使用方法。

### 文档适用对象

- 社区管理员
- 网站运营人员
- 技术维护人员

### 阅读前准备

在阅读本文档前，请确保：
1. 已成功部署彩虹BBS系统
2. 拥有管理员账号（默认：admin/12345678）
3. 了解基本的计算机操作知识

### 术语说明

| 术语 | 说明 |
|------|------|
| 版块 | 讨论区分类，如技术交流、提问求助等 |
| 话题标签 | 帖子的分类标签，用户可自定义 |
| 官方标签 | 管理员推荐的标签，优先显示 |
| 勋章 | 用户成就标识，满足条件自动获得 |
| 信誉分 | 用户信用评分，影响内容展示 |
| 置顶 | 将帖子固定在列表顶部 |
| 精华 | 标记为优质内容的帖子 |

## 第一章 管理后台入门

### 1.1 访问管理后台

管理后台通常部署在独立的端口或路径下。

#### 访问地址

根据部署方式不同，访问地址可能是：
- `http://your-domain.com/admin`
- `http://your-domain.com:8081`（如果使用独立端口）

#### 本地开发环境

如果是本地开发环境：
- 主站通常运行在：`http://localhost:5173`
- 管理后台通常运行在：`http://localhost:5174`（或其他端口）

### 1.2 登录管理后台

#### 登录步骤

1. 打开管理后台访问地址
2. 在登录页面输入：
   - **用户名**：admin（或您的管理员用户名）
   - **密码**：12345678（默认密码，首次登录后请立即修改）
3. 点击「登录」按钮

#### 默认账号

| 账号类型 | 用户名 | 密码 | 说明 |
|----------|--------|------|------|
| 管理员 | admin | 12345678 | 拥有所有权限 |

#### 修改密码

首次登录后，请立即修改管理员密码：

1. 登录管理后台
2. 找到「修改密码」菜单（通常在用户头像下拉菜单中）
3. 输入原密码和新密码
4. 点击「确认」保存

#### 安全提示

⚠️ **重要安全提示**：
- 请勿使用默认密码生产环境
- 定期更换管理员密码
- 不要将管理员账号密码分享给他人
- 使用强密码（包含大小写字母、数字、特殊字符）

### 1.3 管理后台界面概览

#### 左侧导航栏

管理后台采用左侧导航布局，主要菜单包括：

| 菜单项 | 功能说明 |
|--------|----------|
| 仪表板 | 查看统计数据和系统信息 |
| 用户管理 | 管理用户账号、角色、禁言等 |
| 帖子管理 | 管理所有帖子、置顶、删除等 |
| 评论管理 | 管理评论内容 |
| 版块管理 | 管理讨论版块 |
| 标签管理 | 管理话题标签 |
| 公告管理 | 发布和管理公告 |
| 勋章管理 | 管理用户勋章 |
| 投票管理 | 管理投票活动 |
| 举报管理 | 处理用户举报 |
| 防刷配置 | 配置防刷系统参数 |
| 系统设置 | 配置系统功能开关和积分规则 |

#### 顶部栏

- **左侧**：系统Logo和名称
- **右侧**：当前登录用户信息、退出登录按钮

#### 内容区域

- 显示当前选中菜单的功能页面
- 包含表格、表单、按钮等操作元素

## 第三章 用户管理

用户管理模块用于管理社区中的所有用户账号。

### 3.1 用户列表

进入「用户管理」菜单后，显示用户列表表格：

#### 表格列说明

| 列名 | 说明 |
|------|------|
| ID | 用户唯一标识 |
| 用户信息 | 显示用户头像、用户名、邮箱 |
| 角色 | 用户角色（普通用户/版主/管理员） |
| 积分 | 用户积分，可排序 |
| 状态 | 正常/已禁言 |
| 注册时间 | 用户注册日期 |
| 操作 | 角色编辑、禁言/解禁、删除 |

#### 用户角色说明

| 角色 | 标识 | 颜色 | 权限说明 |
|------|------|------|----------|
| 普通用户 | 0 | 灰色 | 基本发帖、评论权限 |
| 版主 | 1 | 橙色 | 可管理指定版块内容 |
| 管理员 | 2 | 红色 | 拥有所有管理权限 |

### 3.2 搜索用户

#### 搜索步骤

1. 在页面右上角的搜索框中输入关键词
2. 关键词可以是用户名或邮箱
3. 点击「搜索」按钮或按回车键执行搜索
4. 搜索结果会实时更新表格

#### 清除搜索

- 点击搜索框右侧的「×」按钮清除搜索关键词
- 或直接删除搜索框中的内容后按回车键

### 3.3 编辑用户角色

#### 修改角色步骤

1. 在用户列表中找到要修改的用户
2. 点击该行的「角色」按钮
3. 在弹出的对话框中选择新角色：
   - 普通用户
   - 版主
   - 管理员
4. 点击「确认」按钮保存

#### 注意事项

⚠️ **重要提示**：
- 只有管理员可以修改用户角色
- 修改角色后立即生效
- 请谨慎授予管理员权限

### 3.4 禁言与解禁

#### 禁言用户

1. 在用户列表中找到要禁言的用户
2. 点击「禁言」按钮
3. 在确认对话框中点击「确定」
4. 用户状态变为「已禁言」

#### 解禁用户

1. 在用户列表中找到已禁言的用户
2. 点击「解禁」按钮
3. 在确认对话框中点击「确定」
4. 用户状态恢复为「正常」

#### 禁言影响

被禁言的用户将无法：
- 发布新帖子
- 发表评论
- 发送私信

但仍可以：
- 浏览帖子
- 点赞和收藏
- 查看个人中心

### 3.5 删除用户

#### 删除步骤

⚠️ **警告**：删除用户是不可逆操作，请谨慎执行！

1. 在用户列表中找到要删除的用户
2. 点击「删除」按钮
3. 仔细阅读确认对话框中的提示：
   > 确定要彻底删除用户 "xxx" 吗？此操作不可恢复，相关帖子、收藏、点赞等数据也将被删除！
4. 确认无误后点击「删除」按钮

#### 删除后果

删除用户将同时删除：
- 用户账号
- 用户发布的所有帖子
- 用户发表的所有评论
- 用户的收藏记录
- 用户的点赞记录
- 用户的关注关系
- 用户获得的勋章

## 第四章 帖子管理

帖子管理模块用于管理社区中的所有帖子内容。

### 4.1 帖子列表

进入「帖子管理」菜单后，显示帖子列表表格：

#### 表格列说明

| 列名 | 说明 |
|------|------|
| ID | 帖子唯一标识 |
| 标题 | 帖子标题，可点击直接在新窗口打开 |
| 作者 | 发帖用户的头像和用户名 |
| 版块 | 帖子所属版块 |
| 浏览/点赞/评论 | 帖子的浏览数、点赞数、评论数 |
| 状态 | 置顶、精华、锁定标识 |
| 发布时间 | 帖子发布日期 |
| 操作 | 置顶/取消置顶、查看、删除 |

#### 状态标签说明

| 标签 | 颜色 | 说明 |
|------|------|------|
| 置顶 | 红色 | 帖子固定在列表顶部 |
| 精华 | 黄色 | 标记为优质内容 |
| 锁定 | 灰色 | 禁止评论 |

### 4.2 搜索帖子

#### 搜索步骤

1. 在页面右上角的搜索框中输入关键词
2. 关键词匹配帖子标题
3. 点击「搜索」按钮或按回车键执行搜索

#### 清除搜索

- 点击搜索框右侧的「×」按钮清除搜索

### 4.3 置顶与取消置顶

#### 置顶帖子

1. 在帖子列表中找到要置顶的帖子
2. 点击「置顶」按钮
3. 在确认对话框中点击「确定」
4. 帖子状态显示「置顶」标签

#### 取消置顶

1. 在帖子列表中找到已置顶的帖子
2. 点击「取消置顶」按钮
3. 在确认对话框中点击「确定」
4. 帖子「置顶」标签消失

#### 置顶规则

- 置顶帖子会显示在版块列表顶部
- 可以同时置顶多个帖子
- 置顶帖子按置顶时间倒序排列

### 4.4 查看帖子

#### 查看步骤

1. 在帖子列表中找到要查看的帖子
2. 点击「查看」按钮，或直接点击帖子标题
3. 帖子会在新标签页中打开

### 4.5 删除帖子

#### 删除步骤

⚠️ **警告**：删除帖子是不可逆操作！

1. 在帖子列表中找到要删除的帖子
2. 点击「删除」按钮
3. 在确认对话框中点击「删除」
4. 帖子从列表中移除

#### 删除后果

删除帖子将同时删除：
- 帖子内容
- 帖子的所有评论
- 帖子的点赞记录
- 帖子的收藏记录
- 帖子的投票（如果有）

## 第五章 评论管理

评论管理模块用于管理社区中的所有评论内容。

### 5.1 评论列表

进入「评论管理」菜单后，显示评论列表表格：

#### 表格列说明

| 列名 | 说明 |
|------|------|
| ID | 评论唯一标识 |
| 内容 | 评论内容，超长内容会截断显示 |
| 作者 | 评论用户的头像和用户名 |
| 所属帖子 | 点击「查看」按钮跳转至原帖子 |
| 点赞 | 评论获得的点赞数 |
| 发布时间 | 评论发布日期 |

- 目标标签的使用次数增加

#### 注意事项

⚠️ **重要提示**：

#### 开关说明

- 关闭：停止新用户注册，仅现有用户可登录

- 开启：用户可以发表评论
- 关闭：所有用户无法评论（管理员除外）

**允许发起投票**：
- 开启：用户可以在帖子中添加投票
- 关闭：用户无法添加投票

**社区维护**：
- 临时关闭注册，只允许老用户访问
- 临时关闭发帖，清理违规内容

**特殊活动**：
- 只允许评论，不允许发新帖
- 限制某些功能

- 评论被删除不扣除积分
- 建议值：1-3分

**签到积分**：
- 用户每日签到获得此积分
- 连续签到额外获得奖励（可在代码中配置）
- 建议值：2-5分

积分可以用于：

- 提升用户等级
- 展示用户成就

## 第十五章 常见问题与故障排除

### 15.1 管理后台问题

#### Q: 无法登录管理后台？

**A**: 请检查：
1. 用户名和密码是否正确
2. 管理后台地址是否正确
3. 后端服务是否正常运行
4. 浏览器控制台是否有错误信息

#### Q: 保存配置后没有生效？

**A**: 请尝试：
1. 刷新管理后台页面
2. 清除浏览器缓存
3. 检查后端日志
4. 确认配置确实保存成功

#### Q: 列表加载很慢？

**A**: 可能原因：
1. 数据量过大，建议分页查询
2. 数据库查询慢，检查索引
3. 网络问题
4. 后端服务负载高

### 15.2 主站问题

#### Q: 用户无法注册？

**A**: 请检查：
1. 系统设置中「允许用户注册」是否开启
2. 用户名是否已存在
3. 邮箱格式是否正确
4. 密码长度是否符合要求

#### Q: 用户无法发帖？

**A**: 请检查：
1. 系统设置中「允许发帖」是否开启
2. 用户是否被禁言
3. 用户信誉分是否过低
4. 是否在频率限制期内

#### Q: 图片上传失败？

**A**: 请检查：
1. 存储配置是否正确（七牛/阿里云/腾讯云）
2. 文件大小是否超限
3. 文件格式是否支持
4. 网络连接是否正常

### 15.3 数据库问题

#### Q: 数据库文件在哪里？

**A**: SQLite 数据库文件通常在：
- 项目根目录下的 `data` 文件夹
- 文件名为 `bbs.db` 或类似

#### Q: 如何备份数据库？

**A**: 直接复制数据库文件：
1. 停止后端服务
2. 复制 `data/bbs.db` 到备份位置
3. 重启后端服务

#### Q: 数据库损坏怎么办？

**A**: 尝试：
1. 从备份恢复
2. 使用 SQLite 工具检查修复
3. 重新初始化（会丢失数据）
