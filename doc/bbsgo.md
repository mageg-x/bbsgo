事情是这样的...

- ✅ 不用这不用那，扔上去就能跑

- ✅ 能快速接入我现有的用户体系

找来找去找不到，索性一拍大腿：**自己写一个！**

## 🛠️ 技术栈（给懂技术的朋友看看）

### 后端
- **语言**：Go
- **框架**：gorilla/mux
- **ORM**：GORM
- **数据库**：SQLite（WAL 模式，性能杠杠的）
- **缓存**：Ristretto（内存缓存）
- **认证**：JWT

### 前端
- **框架**：Vue 3（Composition API）
- **路由**：Vue Router 4
- **状态管理**：Pinia
- **UI 框架**：Element Plus
- **CSS**：Tailwind CSS
- **Markdown**：ByteMD + GFM
- **构建工具**：Vite

## 🚀 2 分钟部署

### 环境要求
- 就这么多！**没有其他依赖**

### 一键启动

前端和管理后台都已经嵌入到二进制程序中，只需要运行一个文件：

```bash
# 方式一：直接运行编译好的二进制
./bbsgo

# 方式二：自己编译
cd server
go mod tidy
go build -o bbsgo main.go
./bbsgo
```

服务会在 `:8080` 端口启动，直接访问即可：
- 主站：http://localhost:8080
- 管理后台：http://localhost:8080/console/login

### 默认账号
- **管理员**：admin / 12345678
- **测试用户**：testuser1 ~ testuser10 / 123456

## ⚙️ 配置说明

**💡 提示：所有配置都在管理后台界面完成，不用写代码！**

登录管理后台（http://localhost:8080/console/login），在「网站设置」页面就能配置：

### 网站配置
- 网站名称
- 是否允许注册
- 是否允许发帖
- 等等...

### 存储配置
支持四种存储方式，在界面上选择即可：
- **本地存储**：直接存在服务器上
- **七牛云**：填写 AccessKey、SecretKey、Bucket、域名
- **阿里云**：填写相关配置
- **腾讯云**：填写相关配置

### 防刷配置
在管理后台的「防刷配置」页面调整：
- 频率限制参数
- 内容质量阈值
- 举报处理规则
- 信誉分系统
