# 彩虹BBS 产品需求文档 (PRD)

## 1. 产品概述

彩虹BBS是一个现代化的社区论坛系统，采用前后端分离架构：
- **前端**：Vue 3 + Element Plus（主站 + 管理后台）
- **后端**：Go + GORM + SQLite
- **特性**：Markdown编辑器、投票系统、勋章成就、私信通知、防刷机制

---

## 2. 用户系统

### 2.1 用户角色与权限

| 角色 | Role值 | 权限说明 |
|------|--------|----------|
| 普通用户 | 0 | 浏览、发帖、评论、点赞、收藏、关注、私信 |
| 版主 | 1 | 管理指定版块的帖子和评论（置顶、删除） |
| 管理员 | 2 | 全站管理、用户管理、配置管理、防刷管理 |

### 2.2 用户属性

```
用户ID、用户名、邮箱、昵称
密码哈希（bcrypt加密）
角色(0/1/2)、头像、个人主页背景、个性签名、个人简介
积分(Credits)、等级(Level)、信誉分(Reputation 0-100)
最后签到时间、注册时间
```

### 2.3 注册与登录

**注册流程：**
1. 填写用户名、邮箱、昵称、密码
2. 系统发送邮箱验证码（60秒倒计时）
3. 验证成功后创建账号

**登录方式：**
- 用户名/密码登录
- JWT Token认证（默认7天过期，可配置）

**管理员可关闭注册功能**（`allow_register` 配置）

## 3. 内容系统

### 3.1 版块 (Forum)

系统预设8个版块：

| ID | 名称 | 描述 |
|----|------|------|
| 1 | 全部 | 默认首页，显示所有版块的帖子 |
| 2 | 技术交流 | 编程语言、框架、架构等纯技术讨论 |
| 3 | 提问求助 | 发帖求助、解答问题 |
| 4 | 业界资讯 | 科技新闻、技术动态、行业趋势 |
| 5 | 资源分享 | 工具、教程、电子书、代码片段 |
| 6 | 求职招聘 | 内推、招聘信息、面经 |
| 7 | 灌水闲聊 | 生活、娱乐、非技术话题 |
| 8 | 站务管理 | 公告、反馈、版务 |

**版块属性：**
- 名称、描述、排序、图标
- 允许发帖标记（可单独关闭）

### 3.2 话题/帖子 (Topic)

**发帖规则：**
- 需填写标题和内容
- 选择所属版块
- 可添加最多3个话题标签
- 支持Markdown编辑（GFM、代码高亮，数学公式）
- 支持上传图片（客户端压缩+秒传检测）
- 支持上传视频（50MB限制）
- 支持附加投票

**帖子属性：**
```
标题、内容(HTML渲染)、作者、所属版块
标签列表（多对多关联）
是否置顶（管理员/作者）
是否锁定、是否精华、是否隐藏
热度评分(HotScore)
点赞数、浏览数、回复数
最后回复时间
是否允许评论
```

**作者置顶：**
- 作者可将自己的帖子置顶（1个）
- 管理员可置顶任意帖子

### 3.3 评论 (Comment)

**评论规则：**
- 支持回复指定评论（`reply_to_id`）
- 楼主可置顶评论
- 楼主可标记"最佳评论"
- 评论被删除时不影响楼层结构

**评论属性：**
```
内容、作者、所属话题
回复目标评论ID（reply_to_id）
点赞数
是否置顶、是否最佳、是否隐藏
```

## 4. 互动系统

### 4.1 点赞 (Like)

- 可点赞帖子或评论
- 取消点赞需再次点击
- 点赞时发送通知

### 4.2 收藏 (Favorite)

- 收藏帖子到个人收藏夹
- 支持取消收藏
- 收藏夹仅本人可见

### 4.3 关注 (Follow)

**关注规则：**
- 关注用户后可查看其最新帖子
- 关注/取消关注操作
- 被关注者获得1积分奖励（`credit_follow`配置）

**关注动态：**
- 展示关注用户的最新帖子
- 按时间倒序排列

### 4.4 分享 (Share)

- 支持分享到：微信、朋友圈、微博、QQ
- 生成分享二维码
- 复制链接

### 4.5 举报 (Report)

**举报原因分类：**
- spam（垃圾广告）
- illegal（违规内容）
- attack（人身攻击）
- rumor（谣言虚假信息）
- other（其他）

**举报处理规则：**
- 用户每日最多举报10次（`max_reports_per_day`配置）
- 不能重复举报同一内容
- 举报3次后自动隐藏内容（`report_threshold`配置）
- 扣除作者5点信誉分
- 7天内被隐藏5次则禁言3天（`report_ban_threshold`、`report_ban_days`配置）

### 5.1 私信 (Message)

- 用户间一对一私信
- 会话列表展示

### 5.2 通知 (Notification)

**通知类型：**
| 类型 | 颜色 | 触发场景 |
|------|------|----------|

| 关注 | 绿色 | 被用户关注 |

## 6. 积分与签到系统

### 6.1 积分 (Credits)

**积分用途：**
- 显示用户活跃度
- 排行榜排序依据

**积分规则（可配置）：**
| 操作 | 积分 |
|------|------|
| 签到 | +10（`credit_signin`配置） |
| 连续签到 | +15（`credit_signin_consecutive`配置） |
| 被关注（被关注者获得） | +1（`credit_follow`配置） |

### 6.2 签到 (SignIn)

**签到规则：**
- 每日签到1次
- 连续签到获得更多积分
- 签到状态持久化

**签到配置：**
- `credit_signin`: 基础签到积分（默认10）
- `credit_signin_consecutive`: 连续签到积分（默认15）

## 7. 勋章系统

### 7.1 勋章类型

| 类型 | 说明 | 勋章 |
|------|------|------|
| basic | 基础入门 | 初来乍到、首次发声、首次评论 |
| advanced | 进阶成就 | 笔耕不辍(50帖)、社区活宝(1000评)、广受欢迎(1000赞)、金牌评论(5次最佳) |
| top | 顶级荣耀 | 意见领袖(500粉)、社区传奇(2年+200帖+500赞+10最佳) |

### 7.2 勋章列表

| 名称 | 图标 | 条件 | 类型 | 条件类型 |
|------|------|------|------|----------|
| 初来乍到 | newcomer | 注册即获得 | basic | register |
| 首次发声 | first-post | 发布第1个帖子 | basic | topic_count=1 |
| 热心回复 | first-comment | 发布第1条评论 | basic | comment_count=1 |
| 笔耕不辍 | writer | 发布50个帖子 | advanced | topic_count=50 |
| 社区活宝 | community-star | 发布1000条评论 | advanced | comment_count=1000 |
| 广受欢迎 | popular | 累计获得1000个点赞 | advanced | like_count=1000 |
| 金牌评论 | gold-comment | 获得5次最佳评论 | advanced | best_comment=5 |
| 意见领袖 | opinion-leader | 被500个用户关注 | top | follower_count=500 |
| 社区传奇 | legend | 注册满2年+发帖≥200+获赞≥500+最佳评论≥10 | top | combination |

### 7.3 勋章授予规则

**勋章检查触发时机：**
- 用户注册时检查register类型勋章
- 发帖后检查topic_count类型勋章
- 评论后检查comment_count类型勋章
- 被点赞后检查like_count类型勋章
- 评论被标记最佳后检查best_comment类型勋章
- 被关注后检查follower_count类型勋章

**组合条件（社区传奇）：**
```json
{
  "register_days": 730,
  "topic_count": 200,
  "like_count": 500,
  "best_comment": 10
}
```
需同时满足所有条件。

### 7.4 勋章陈列馆

- 展示用户所有勋章（含未获得）
- 按类型分类显示（基础入门/进阶成就/顶级荣耀）
- 显示获得进度
- 显示获得时间
- 支持撤销勋章（管理员）

## 8. 投票系统

### 8.1 投票属性

```
关联话题(TopicID)
投票标题、投票描述
投票类型(single单选/multiple多选)
最大可选数(MaxChoices)
截止时间(EndTime)
是否已结束(IsEnded)
总投票数(TotalVotes)
选项列表([]PollOption)
```

### 8.2 创建投票

**规则：**
- 帖子作者或管理员可为帖子创建投票
- 每个帖子只能创建一个投票
- 选项数量：2-10个
- 单选投票只能选择1个选项
- 多选投票可设置最大可选数
- 截止时间可选，不设置则无期限

**权限：**
- 帖子作者可创建投票
- 版主/管理员可创建投票

### 8.3 参与投票

**规则：**
- 每个用户每个投票只能投票一次
- 单选投票只能选择1个选项
- 多选投票选择数量不能超过MaxChoices
- 投票截止后不能投票
- 投票后显示已投选项和当前结果

**返回信息：**
- 投票信息（含选项列表）
- 是否已投票
- 已投选项ID列表
- 各选项投票数和百分比

### 8.4 投票管理（管理员）

- 查看投票列表
- 结束投票
- 删除投票
- 修改投票（不能缩短截止时间）

## 9. 信誉分系统

### 9.1 信誉分范围

- **初始值**：100分
- **范围**：0-100分

### 9.2 信誉分等级

| 信誉分 | 等级 | 说明 |
|--------|------|------|
| 80-100 | normal | 正常用户 |
| 60-79 | captcha | 需验证码 |
| 40-59 | limited | 受限（每天3帖10评） |
| 20-39 | restricted | 高度受限 |
| 0-19 | banned | 被禁言 |

### 9.3 信誉分变化规则

| 操作 | 变化 | 说明 |
|------|------|------|
| 发布低质量内容 | -1 | 被内容质量检测拦截 |
| 内容被举报并隐藏 | -5 | 举报处理自动执行 |
| 每日恢复 | +1 | 30天内无违规自动恢复 |
| 恶意举报 | -1 | 举报被判定为恶意 |

### 9.4 信誉分禁言

- 信誉分低于20时自动禁言（`ban_reputation_threshold`配置）
- 信誉分恢复到20以上自动解除禁言
- 禁言类型为"reputation"

## 10. 防刷系统

### 10.1 频率限制

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| topic_min_interval | 60秒 | 发帖最小间隔 |
| comment_min_interval | 30秒 | 评论最小间隔 |
| max_topics_per_day | 10 | 每日最大发帖数 |
| max_comments_per_day | 50 | 每日最大评论数 |
| new_user_max_topics_per_day | 3 | 新用户每日最大发帖数 |
| new_user_max_comments_per_day | 10 | 新用户每日最大评论数 |
| new_user_hours | 24小时 | 新用户定义时间 |

**附加规则：**
- 信誉分<40时，日上限降为3
- 新用户（注册24小时内）使用更严格限制
- 1小时内相同内容哈希不能重复发布（重复内容检测）

### 10.2 内容质量检测

**检测项目：**
| 检测项 | 判定 | 处理 |
|--------|------|------|
| 空内容 | 无实质信息 | 拒绝发布 |
| 纯符号 | 仅符号或表情 | 拒绝发布 |
| 内容太短 | 少于10字符（汉字+字母+标点都计入） | 拒绝发布，显示"内容太短，最少需要X个字符" |
| 重复字符 | 5个以上连续重复字符 | 拒绝发布 |
| 广告关键词 | 含"加VX"、"加微信"、"赚钱"等 | 拒绝发布 |
| 外部链接过多 | 3个以上非白名单外部链接 | 警告，扣分 |

**白名单域名：**
- localhost
- 127.0.0.1
- bbsgo.com

### 10.3 举报处理

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| report_threshold | 3 | 自动隐藏阈值 |
| report_ban_threshold | 5 | 自动禁言阈值（7天内） |
| report_ban_days | 3 | 禁言天数 |
| max_reports_per_day | 10 | 每日最大举报次数 |

### 10.4 热度评分

**计算公式：**
```
基础得分 = (点赞数×2 + 评论数×1.5 + 浏览数×0.1) / 时间衰减
最终得分 = 基础得分 × 内容质量系数 × 用户信誉系数
```

**衰减规则：**
- 使用平方根衰减，越老的话题衰减越慢

**惩罚系数：**
- 低质量内容：×0.3（`low_quality_hot_multiplier`配置）
- 低信誉分用户(<60)：×0.5（`low_reputation_hot_multiplier`配置）

### 10.5 防刷配置页面

管理后台提供5个标签页配置：

1. **频率限制**：发帖/评论最小间隔、单日最大发帖/评论数、新用户限制
2. **内容质量**：最小内容长度、重复字符阈值、内容相似度阈值
3. **信誉分系统**：低信誉分阈值、禁言阈值、启用低信誉分禁言、热度权重
4. **举报处理**：自动隐藏阈值、自动禁言阈值、自动禁言天数、每日举报上限
5. **用户信誉**：用户信誉分列表、调整信誉分、禁言/解禁用户
